# Stick Cross Bridge

Stick Cross Bridge adalah endless game dua dimensi yang ingin menyebrangi jurang.

### Story Line
Stick Cross Bridge adalah game endless di mana pemain dituntut harus menyeberangi jurang.

### Gameplay
Game ini bertujuan untuk seberapa jauh pulau yang dikunjungi dan dilewati demi meraih skor setinggi-tingginya. Player dituntut menyeberangi jurang dengan sebuah tongkat yang panjangnya bisa dibuat berdasarkan lamanya menekan tombol action. Player akan jatuh apabila panjang dari tongkat melebihi atau kurang dari lebar pulau.

### Character
1. Player

### Concept and Environment
Permainan sederhana yang ketika dibuka akan muncul jendela awal dengan tombol play anda bisa menikmati game yang dibuat. Player akan menyeberangi jurang apabila jembatan sesuai dengan ukuran pulau selanjutnya.

### Action
- Pemain akan menyeberangi jurang menggunkan tongkat
- Pemain akan menekan klik kiri pada mouse untuk mendapatkan tongkat jembatan
- Pemain tidak bisa menyesuaikan panjang tongkat apabila sudah terlalu panjang
- Pemain akan jatuh ketika panjang tongkat tidak sesuai dengan pulau
